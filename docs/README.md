# Documentation

This directory holds the architecture, service boundaries, API contracts, and RFCs for the Project Zomboid launcher monorepo.

- `vision.md` — core vision and product goals
- `architecture.md` — system architecture and component layout
- `service-boundaries.md` — service responsibilities
- `api-contracts.md` — defined APIs for services and agents
- `manifest-schema.md` — manifest structure and content rules
- `profile-system.md` — profile isolation design
- `roadmap.md` — phased implementation plan
- `rfc/` — RFCs for architecture and contract decisions
